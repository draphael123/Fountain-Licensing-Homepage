# State-by-State Rules

This section documents licensing requirements by state for all credential types Fountain Vitality operates under. Use the matrix for a quick cross-reference, then drill into each credential type for detailed requirements.

## How to Use This Section

1. **Start with the [State Requirements Matrix](state-matrix.md)** — quick lookup for any state + credential combo
2. **Drill into the credential-specific pages** for full requirements, fees, application links, and renewal info
3. **Check the [Compliance Calendar](../compliance-calendar/README.md)** for upcoming renewals in each state

## Important Flags

> **🔴 Non-Compact State** — Full application required; no reciprocity shortcut
>
> **🟡 Compact Member** — May qualify for expedited or multi-state license
>
> **🟢 Telehealth Exception** — State has a telehealth-specific registration or exemption
