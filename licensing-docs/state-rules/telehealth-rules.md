# Telehealth Prescribing Rules by State

## Ryan Haight Act & DOGE Exemptions

The Ryan Haight Act requires an in-person evaluation before prescribing controlled substances via telehealth. COVID-era DEA exemptions extended telemedicine prescribing authority for Schedule III-V substances (including testosterone and other HRT/TRT medications).

> **Note:** Confirm current exemption status with compliance before prescribing controlled substances via telehealth in new states.
