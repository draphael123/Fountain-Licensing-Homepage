# RN Licensing Requirements by State

## eNLC Compact

Registered Nurses holding an active eNLC compact license may practice in any eNLC member state without an additional state license.

The eNLC covers the same states as the NP compact. See [NP Requirements](np-requirements.md) for the full state list.

---

## Standard RN Application Process

1. Submit state application + fee to nursing board
2. Provide proof of active compact license or home state RN license
3. Pass NCLEX-RN (first-time applicants)
4. Complete state-specific jurisprudence exam if required
5. Register with state PMP if prescribing

---

## Renewal

Most states require RN license renewal every 2 years. CE requirements vary:

| State Example | CE Hours Required | CE Type |
|---|---|---|
| Florida | 24 hours / 2 years | General + specific topics |
| Texas | 20 hours / 2 years | General |
| California | 30 hours / 2 years | General |

See [CEU Tracker](https://ceu-tracker.vercel.app/) for current CE completion status.
