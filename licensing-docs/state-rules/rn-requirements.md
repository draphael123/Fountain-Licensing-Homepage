# 🏥 RN Licensing Requirements

## eNLC Compact

{{% hint style="success" %}}
RNs holding an active eNLC license can practice in any member state without an additional state license. The eNLC covers the same states as the NP compact — see [NP Requirements](np-requirements.md) for the full list.
{{% endhint %}}

---

## Standard Application Process

1. Submit state application + fee to nursing board
2. Provide proof of active compact or home-state RN license
3. Pass NCLEX-RN (first-time applicants only)
4. Complete state jurisprudence exam if required
5. Register with state PMP if applicable

---

## CE Requirements for RN Renewal

Most states require renewal every 2 years. CE requirements vary — see [CE Requirements](../trackers/ce-requirements.md) for the full breakdown.

| Example State | CE Hours / 2 Years | Special Topics |
|---|:---:|---|
| Florida | 24 | 2hr HIV, 2hr domestic violence |
| Texas | 20 | 2hr jurisprudence |
| California | 30 | Pain management, end-of-life |
