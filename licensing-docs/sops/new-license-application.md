# SOP: New License Application

**Trigger:** A new provider is joining Fountain Vitality and needs a license in one or more states, OR Fountain is expanding into a new state and existing providers need licenses there.

**Owner:** Licensing Team (Faith, Tammy, Brittany)
**Approver:** Daniel Raphael (Clinical Operations)
**Timeline:** Begin 90 days before provider's intended start date or state go-live

---

## Pre-Application Checklist

Before starting any application, confirm the following with the provider:

- [ ] Full legal name (as it appears on government ID)
- [ ] Date of birth
- [ ] SSN (for background check / identity verification)
- [ ] NPI number (Type 1)
- [ ] Home address and practice address
- [ ] Current licenses held (copy of each)
- [ ] DEA number(s) currently held
- [ ] Education and training verification (school, graduation date, transcript)
- [ ] Board exam scores (NCLEX, USMLE, etc.)
- [ ] Certifications (ANCC, AANP, board certifications)
- [ ] Malpractice history (any claims, disciplinary actions)
- [ ] Background check authorization (if required by state)
- [ ] Collaborative physician info (if required in state)

---

## Application Steps

### Step 1: Identify Requirements
1. Look up the target state in the [State Requirements Matrix](../state-rules/state-matrix.md)
2. Navigate to the appropriate credential page (NP / MD-DO / RN)
3. Note: fee, required documents, application portal URL, processing time

### Step 2: Create Application Account
1. Go to the state board's online portal
2. Create a new applicant account (or log in if previously registered)
3. Save login credentials to the team's secure password manager

### Step 3: Complete Application
1. Fill in all required fields — use the provider's profile data from Notion
2. Upload all required documents (transcripts, certifications, photo ID, etc.)
3. Request primary source verifications if required (e.g., NURSYS for nursing licenses)
4. Submit collaborating physician agreement if required (send to physician for signature via DocuSign)

### Step 4: Pay Application Fee
1. Process payment via Fountain's designated payment method
2. Save receipt and record fee in expense tracker

### Step 5: Track Application Status
1. Note the application/reference number in Notion Provider Dashboard
2. Check status every 5–7 business days
3. Respond promptly to any board deficiency notices

### Step 6: License Issued
1. Download and save the license certificate to provider's Drive folder
2. Update Notion Provider Dashboard with:
   - License number
   - Issue date
   - Expiration date
   - State
3. Notify Daniel Raphael via Slack: license issued, provider/state/expiration
4. If license requires CSR or DEA, initiate [CSR Application](csr-application.md) and [DEA Registration](dea-registration.md) immediately

---

## Escalation

If an application is delayed beyond expected processing time, or if the board issues a deficiency:

1. Contact the board directly by phone
2. Document the call (date, rep name, outcome) in Notion
3. Escalate to Daniel if issue cannot be resolved within 48 hours

---

## Common Pitfalls

- Missing primary source verification (NURSYS, FCVS) — these take time; request early
- Incorrect practice address on application (must match DEA registration)
- Forgetting to send collaborative agreement to physician before submission
- Submitting application before background check is cleared
