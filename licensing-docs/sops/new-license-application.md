# SOP: New State License Application

**Owner:** Ashley Gwinn / Licensing Team
**Trigger:** New state expansion or new provider hire requiring additional state license

---

## Steps

### 1. Pre-Application Verification
- [ ] Confirm state is on expansion plan (check with CCOO/CMO)
- [ ] Verify provider credential type (NP, MD/DO, RN) and compact eligibility
- [ ] Check if state is eNLC/IMLC compact member → if yes, use [Compact SOP](compact-license.md)
- [ ] Pull provider's current license list from Provider & Compliance Dashboard

### 2. Gather Required Documents
- [ ] Proof of national certification (ANCC, AANP, or board cert for MDs)
- [ ] Transcripts (some states require official sealed transcripts)
- [ ] Current state license copy
- [ ] NPI number
- [ ] Malpractice insurance certificate (NFP COI)
- [ ] DEA certificate (if required for state)

### 3. Application Submission
- [ ] Open state board portal (see [Board Directory](../reference/board-directory.md))
- [ ] Complete online application
- [ ] Pay application fee (see [Fee Reference Table](../reference/fee-table.md))
- [ ] Upload all required documents
- [ ] Note confirmation/application number in Provider & Compliance Dashboard

### 4. Post-Submission
- [ ] Update Provider & Compliance Dashboard with application date and status = Pending
- [ ] Set 30-day follow-up reminder in Google Calendar
- [ ] Notify provider via email with application confirmation

### 5. License Received
- [ ] Upload license copy to provider's DocuSign file / Drive folder
- [ ] Update Provider & Compliance Dashboard with license number and expiration
- [ ] Add to License Expiration Monitor
- [ ] Apply for state CSR if required (see [CSR SOP](csr-application.md))
- [ ] Register with state PMP
- [ ] Notify CCOO/CMO and scheduling team that provider is live in state

---

**Average timeline:** 4–16 weeks depending on state. Compact states: 2–4 weeks.
