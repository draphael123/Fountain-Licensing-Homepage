# 📄 New State License Application

**Owner:** Ashley Gwinn / Licensing Team
**Trigger:** New state expansion or new provider hire

{% hint style="info" %}
**Before starting:** Check if the state is eNLC or IMLC compact-eligible. If yes, use the [Compact License SOP](compact-license.md) instead — it's faster and cheaper.
{% endhint %}

---

## Phase 1 — Pre-Application

- [ ] Confirm state is on the expansion plan (check with CCOO/CMO)
- [ ] Verify provider credential type (NP / MD / DO / RN) and compact eligibility
- [ ] Pull provider's current license list from the Provider & Compliance Dashboard
- [ ] Confirm malpractice insurance is active (NFP COI)

## Phase 2 — Gather Documents

- [ ] National certification proof (ANCC, AANP, or board cert for MDs)
- [ ] Official transcripts (check if state requires sealed copies)
- [ ] Copy of current state license
- [ ] NPI number
- [ ] DEA certificate (if required in new state)
- [ ] Malpractice COI from NFP

## Phase 3 — Submit Application

- [ ] Open state board portal → [Board Directory](../reference/board-directory.md)
- [ ] Complete online application
- [ ] Pay fee → [Fee Reference Table](../reference/fee-table.md)
- [ ] Upload all documents
- [ ] Record application number + submission date in Provider & Compliance Dashboard
- [ ] Set 30-day follow-up reminder

## Phase 4 — License Received

- [ ] Upload license to provider's Drive folder
- [ ] Update Provider & Compliance Dashboard (license #, expiration)
- [ ] Add to [License Expiration Monitor](https://license-expiration-monitor-v2-3ykw-kbikc243m.vercel.app/)
- [ ] Apply for CSR if required → [CSR SOP](csr-application.md)
- [ ] Register with state PMP
- [ ] Notify CCOO/CMO and scheduling that provider is live in state

{% hint style="warning" %}
**Timeline:** 4–16 weeks for standard applications. Compact states: 2–4 weeks. Set calendar reminders accordingly.
{% endhint %}
