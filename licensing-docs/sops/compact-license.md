# SOP: Compact License (eNLC / IMLC)

**Owner:** Ashley Gwinn / Licensing Team

---

## eNLC (Nurse Licensure Compact) — for NPs and RNs

### Eligibility Check
- [ ] Provider's home state is an eNLC member state
- [ ] Provider has active, unencumbered home state license

### Process
1. No additional application needed for eNLC states — compact privileges are automatic
2. Confirm target state is on the [eNLC member list](../state-rules/np-requirements.md)
3. Update Provider & Compliance Dashboard to reflect compact coverage

---

## IMLC (Interstate Medical Licensure Compact) — for MDs/DOs

### Eligibility Check
- [ ] Provider holds full, unrestricted license in a principal state
- [ ] No disciplinary history
- [ ] Board certified or eligible

### Process
1. Apply through the [IMLC portal](https://imlcc.org/)
2. Select participating states
3. Pay per-state fee (varies, typically $200–$700/state)
4. Timeline: 2–4 weeks per state
5. Update Provider & Compliance Dashboard for each approved state

---

**Important:** Compact privileges do not automatically include DEA or CSR — those must be obtained separately per state.
