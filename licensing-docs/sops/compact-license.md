# SOP: Compact License (eNLC / IMLC / APRN Compact)

Compact licensure programs allow eligible providers to obtain multi-state or expedited licenses more efficiently than traditional individual state applications.

---

## eNLC — Enhanced Nurse Licensure Compact (RNs)

### What It Is
The eNLC allows RNs whose **home state** is a compact member to practice in other compact member states without obtaining a separate license.

### When to Use This SOP
- New RN hire whose home state is an eNLC member
- Fountain is expanding to a new eNLC state and an RN already holds a compact license

### Steps
1. Confirm provider's home state is an eNLC compact member ([ncsbn.org](https://www.ncsbn.org))
2. Confirm the target state is also an eNLC compact member
3. Verify the provider's home state license is **active and in good standing**
4. If both states are compact members: **no new license application needed** — document in Notion
5. If target state is NOT compact: proceed with standard [New License Application SOP](new-license-application.md)

### Compact License Verification
- Verify via [NURSYS](https://www.nursys.com) — this shows whether the license is a compact (multi-state) license

## IMLC — Interstate Medical Licensure Compact (MDs / DOs)

### What It Is
The IMLC allows physicians to apply for expedited licenses in member states through a streamlined, centralized process.

### When to Use This SOP
- Physician (MD or DO) needs to be licensed in a new IMLC member state
- Provider is eligible for IMLC (has a designated State of Principal License in an IMLC state)

### Eligibility Requirements
- Hold a full, unrestricted medical license in the State of Principal License (SPL)
- SPL state must be an IMLC member
- No disciplinary actions, criminal history, or pending investigations
- Board-certified or met specific postgraduate training requirements

### Steps
1. Confirm both the SPL state and target state are IMLC members ([imlcc.org](https://www.imlcc.org))
2. Log into the IMLC portal: [imlcc.org](https://www.imlcc.org)
3. Complete IMLC application:
   - Select target state(s)
   - Designate SPL
   - Upload required documentation
4. Pay IMLC fee: approximately $700 for initial application + per-state fees
5. Processing time: typically 30–60 days (vs. 3–6 months for standard)
6. License issued by individual state board — download and update Notion

## APRN Compact (NPs)

### What It Is
The APRN Compact (separate from eNLC) allows APRNs to hold one multi-state APRN license. As of 2025, fewer states have enacted this compact compared to eNLC.

### Steps
1. Check if provider's home state has enacted the APRN Compact
2. Check if target state is an APRN Compact member
3. If applicable, apply via the home state board using the compact endorsement process
4. If not applicable, proceed with standard [New License Application SOP](new-license-application.md)

## Decision Tree

```
Provider needs to practice in a new state
         │
         ▼
Is the provider an RN?
  ├── Yes → Is home state eNLC compact member?
  │           ├── Yes + target state also compact → No new license needed ✅
  │           └── No → Full RN license application
  │
  └── No → Is the provider an MD/DO?
             ├── Yes → Is provider eligible for IMLC? Is target state IMLC?
             │           ├── Yes → Use IMLC expedited process ✅
             │           └── No → Full MD/DO license application
             │
             └── NP → Check APRN Compact eligibility, otherwise full NP application
```
