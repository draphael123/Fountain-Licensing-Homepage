# ✅ Provider Onboarding

**Owner:** Daniel Raphael / Ashley Gwinn
**Trigger:** New provider hired and background check cleared

---

## 🗓️ Phase 1 — Week 1–2: Pre-Start

- [ ] **DocuSign:** Send provider agreement
- [ ] **Gusto:** Set up payroll profile
- [ ] **Email:** Create Fountain email address
- [ ] **NFP:** Notify to add provider to malpractice policy
- [ ] **CE Broker:** Set up provider account

{{% hint style="info" %}}
DocuSign and Gusto must be completed before the provider's first payroll period.
{{% endhint %}}

## 🏛️ Phase 2 — Weeks 2–8: Licensing

- [ ] Apply for all required state licenses → [New License SOP](new-license-application.md)
- [ ] Apply for DEA in required states → [DEA SOP](dea-registration.md)
- [ ] Apply for CSR where required → [CSR SOP](csr-application.md)
- [ ] Register with state PMPs → [PMP Tracker](../trackers/pmp-tracker.md)
- [ ] Execute CPA with collaborating physician if required → [CPA Tracker](../trackers/cpa-tracker.md)

## 💻 Phase 3: Systems Setup

- [ ] **OnceHub:** Configure booking page with availability
- [ ] **Doxy:** Set up virtual visit room
- [ ] **Intercom:** Add to support platform
- [ ] **Pharmacy portals:** Register at Empower, TPH, Curexa, Belmar

## 🚀 Phase 4: Go-Live

- [ ] Confirm all required licenses are active
- [ ] Notify CCOO, CMO, and scheduling team
- [ ] Add to patient-facing state booking pages

{{% hint style="warning" %}}
Provider cannot see patients in a state until the license for that state is fully active in the system.
{{% endhint %}}
