# SOP: Provider Onboarding

**Owner:** Daniel Raphael / Ashley Gwinn
**Trigger:** New provider hired and background check cleared

---

## Phase 1 — Pre-Start (Weeks 1–2)
- [ ] DocuSign: Send provider agreement
- [ ] Gusto: Set up payroll profile
- [ ] Email: Create Fountain email address
- [ ] Notify NFP (malpractice insurance) to add provider to policy
- [ ] Begin license verification in all required states

## Phase 2 — Licensing (Weeks 2–8)
- [ ] Apply for all required state licenses (see [New License SOP](new-license-application.md))
- [ ] Apply for DEA in all required states (see [DEA SOP](dea-registration.md))
- [ ] Apply for CSR where required (see [CSR SOP](csr-application.md))
- [ ] Register with state PMPs
- [ ] Execute CPA with collaborating physician if required

## Phase 3 — Systems Setup
- [ ] OnceHub: Configure booking page with availability
- [ ] Doxy: Set up virtual visit room
- [ ] Intercom: Add to support platform
- [ ] Pharmacy portals: Register at Empower, TPH, Curexa, Belmar
- [ ] CE Broker: Set up account

## Phase 4 — Go-Live
- [ ] Confirm all required licenses are active
- [ ] Notify CCOO, CMO, and scheduling team
- [ ] Provider added to patient-facing state booking pages
