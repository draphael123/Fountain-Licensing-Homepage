# SOP: License Renewal

**Trigger:** A provider's license is approaching expiration (within 90 days) or the compliance calendar flags an upcoming renewal.

**Owner:** Licensing Team (Faith, Tammy, Brittany)
**Approver:** Daniel Raphael (Clinical Operations)
**Timeline:** Begin process **60–90 days before expiration**

---

## Renewal Trigger Sources

- Notion Provider & Compliance Dashboard expiration alerts
- Compliance Calendar 90-day rolling view
- CE Broker CE completion tracking
- State board renewal reminder emails (forward to licensing team inbox)

---

## Renewal Checklist

### Step 1: Confirm CE Completion
1. Log into CE Broker (or check provider's CE records)
2. Verify the required CE hours are completed for the renewal period
3. Check for mandatory CE topics (e.g., FL: medical errors, opioid prescribing; TX: jurisprudence)
4. If CE is incomplete: notify provider immediately with a deadline and course recommendations

### Step 2: Open Renewal Application
1. Log into the state board's renewal portal
2. Select the provider's license for renewal
3. Confirm all information is current (address, NPI, certifications)
4. Update any changes (name change, practice address change, etc.)

### Step 3: Attestations
1. Complete required attestations (no disciplinary actions, malpractice disclosures, etc.)
2. If there are disclosures to make, escalate to Daniel and Lindsay before submitting

### Step 4: Upload CE Documentation (if required)
1. Some states require CE certificates to be uploaded
2. Obtain CE certificates from provider or CE Broker
3. Upload to renewal application

### Step 5: Pay Renewal Fee
1. Process payment using Fountain's payment method
2. Save receipt and log in expense tracker

### Step 6: Submit & Confirm
1. Submit the renewal application
2. Note the confirmation/reference number in Notion
3. Monitor for renewal certificate issuance (typically 1–30 days depending on state)

### Step 7: Update Records
1. Download renewed license certificate
2. Update Notion Provider Dashboard:
   - New expiration date
   - Renewal confirmation number
3. Save renewed license to provider's Drive folder
4. Notify Daniel via Slack: renewal complete, new expiration date

---

## Late Renewals / Expired Licenses

> ⚠️ **An expired license means the provider cannot practice in that state.** Notify Daniel immediately if a license expires or is at risk of lapsing.

If a license has already expired:
1. Check if the state allows late renewal (grace period varies by state)
2. Some states require reinstatement application vs. renewal — check state board rules
3. Notify Daniel and CMO — provider must stop seeing patients in that state until reinstated
4. Expedite reinstatement application and document timeline

---

## Renewal Tracking

All renewals should be tracked in the Notion Provider Dashboard with:
- Renewal started date
- CE verified date
- Submitted date
- New expiration date
