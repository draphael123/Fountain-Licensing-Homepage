# SOP: CSR Application (Controlled Substance Registration)

**Trigger:** A provider needs to prescribe controlled substances in a new state, or their existing CSR is expiring.

**Owner:** Licensing Team
**Timeline:** Apply immediately after state professional license is issued

---

## Prerequisites

Before applying for a CSR:
- [ ] Provider holds an **active state professional license** in the target state
- [ ] Provider has (or is applying for) a **DEA registration** for that state
- [ ] Provider's NPI is active
- [ ] Practice address confirmed

---

## Application Steps

### Step 1: Identify the Issuing Agency
1. Look up the state in [CSR & DEA Requirements](../state-rules/csr-dea-requirements.md)
2. Identify the correct agency (e.g., Florida DOH, Texas DPS, Georgia GDNA)
3. Navigate to the agency's portal

### Step 2: Complete the CSR Application
1. Create or log in to the agency portal
2. Enter provider's personal info, license number, NPI, DEA number
3. Enter practice address (must match DEA registration)
4. Complete any required attestations

### Step 3: Pay CSR Fee
1. Fees typically range $25–$75 depending on state
2. Process payment and save receipt

### Step 4: Submit & Track
1. Submit application and note confirmation number in Notion
2. Processing times vary: 1–4 weeks typical
3. Follow up if no response after 2 weeks

### Step 5: CSR Issued
1. Download CSR certificate
2. Update Notion Provider Dashboard with:
   - CSR number
   - Issue date
   - Expiration date
3. Save certificate to provider's Drive folder
4. Notify Daniel via Slack

---

## CSR Renewal

CSRs typically renew every 2 years, often aligned with the professional license. Use the same process above for renewal, initiated 60 days before expiration.
