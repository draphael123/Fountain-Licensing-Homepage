# 💊 CSR Application

**Owner:** Ashley Gwinn / Licensing Team
**Trigger:** Provider receives new state license in a CSR-required state

---

## Step 1 — Confirm CSR is Required

- [ ] Check [CSR/DEA Requirements](../state-rules/csr-dea-requirements.md)
- [ ] Confirm provider has active DEA registration

{% hint style="warning" %}
CSR is required in 21 states. Missing it is a compliance violation — confirm before the provider begins prescribing.
{% endhint %}

## Step 2 — Gather Documents

- [ ] Copy of active state license
- [ ] Copy of DEA certificate
- [ ] NPI number
- [ ] Malpractice insurance COI

## Step 3 — Apply

- [ ] Navigate to state DEA/pharmacy board portal
- [ ] Complete CSR application + pay fee → [Fee Table](../reference/fee-table.md)
- [ ] Record application number in Provider & Compliance Dashboard

## Step 4 — Receipt

- [ ] Upload CSR certificate to provider's Drive folder
- [ ] Update Provider & Compliance Dashboard
- [ ] Set renewal reminder (most CSRs renew annually)
