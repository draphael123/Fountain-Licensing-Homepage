# 🏛️ DEA Registration

**Owner:** Ashley Gwinn / Licensing Team
**Trigger:** New provider hire, new state expansion, or DEA renewal due

---

## New Registration

### Pre-Application
- [ ] Confirm provider has active state license in target state
- [ ] Confirm provider does not already have DEA for that state
- [ ] Have ready: state license copy, NPI, malpractice COI

### Application
- [ ] Go to [DEA Online Registration](https://www.dea.diversion.usdoj.gov/)
- [ ] Select **New Registration → Practitioner**
- [ ] Complete Form DEA-224
- [ ] Pay **$888** registration fee (3-year term)
- [ ] Record application number

### After Approval (4–6 weeks)
- [ ] Update Provider & Compliance Dashboard (DEA number + expiration)
- [ ] Apply for state CSR if required → [CSR SOP](csr-application.md)

---

## DEA Renewal

{% hint style="info" %}
DEA sends a renewal reminder 60 days before expiration. Renewal fee: **$888** for 3 years.
Renewal URL: [dea.diversion.usdoj.gov](https://www.dea.diversion.usdoj.gov/online_forms_and_applications/renewal.htm)
{% endhint %}

- [ ] Receive 60-day reminder email from DEA
- [ ] Complete renewal online
- [ ] Update Provider & Compliance Dashboard with new expiration date
