# SOP: DEA Registration

**Trigger:** A provider needs a DEA number in a new state to prescribe controlled substances.

**Owner:** Licensing Team
**Timeline:** Apply after state professional license and CSR (if required) are issued

---

## Prerequisites

- [ ] Active state professional license in target state
- [ ] State CSR obtained (if state requires it before DEA)
- [ ] Active existing DEA in at least one state (for practitioners already registered)
- [ ] Practice address confirmed

---

## Application Steps

### Step 1: Go to DEA Diversion Portal
- URL: [deadiversion.usdoj.gov](https://www.deadiversion.usdoj.gov/webforms/dtStartRegistration.jsp)
- If provider already has a DEA: log in and add a new state registration
- If provider has never had a DEA: complete new practitioner registration

### Step 2: Complete the Application
- Practitioner type: select the appropriate category (NP, MD, DO, RN-APRN, etc.)
- Enter state professional license number
- Enter state CSR number (if applicable)
- Enter NPI number
- Enter practice address (must match CSR and state license)
- Select schedules: typically **III, IV, V** for Fountain providers (confirm with CMO if Schedule II needed)

### Step 3: Pay DEA Fee
- Fee: **$888 per state** (3-year registration, as of 2025)
- Payment by credit card or check
- Save receipt and log in expense tracker

### Step 4: Submit & Track
1. Note the application tracking number in Notion
2. DEA processing typically takes **4–6 weeks**
3. Check status via DEA portal or call: 1-800-882-9539

### Step 5: DEA Issued
1. Download DEA registration certificate
2. Update Notion Provider Dashboard:
   - DEA number
   - State
   - Issue date
   - Expiration date (3 years from issue)
3. Save certificate to provider's Drive folder
4. Notify Daniel via Slack

---

## DEA Renewal

DEA registrations are valid for 3 years. DEA sends renewal notices ~60 days before expiration. Renew online at the DEA Diversion portal.

---

## Important Notes

- A separate DEA number is required for **each state**
- DEA number format: [Letter][Letter][7 digits] (e.g., AB1234567)
- If a provider leaves Fountain, their DEA registration should be updated with their new practice address — coordinate with offboarding process
- Lost DEA certificate: request duplicate via DEA portal
