# SOP: DEA Registration

**Owner:** Ashley Gwinn / Licensing Team
**Trigger:** New provider hire, new state expansion, or DEA renewal due

---

## New Registration

### 1. Pre-Application
- [ ] Confirm provider has active state license in target state
- [ ] Confirm provider does not already have DEA for that state
- [ ] Gather: state license copy, NPI, malpractice COI, SSN (restricted access)

### 2. Application
- [ ] Go to [DEA Online Registration](https://www.dea.diversion.usdoj.gov/)
- [ ] Select "New Registration" → Practitioner
- [ ] Complete Form DEA-224
- [ ] Pay $888 fee (3-year registration)
- [ ] Note application number

### 3. Processing
- DEA typically processes in 4–6 weeks
- [ ] Update Provider & Compliance Dashboard with DEA number and expiration when received
- [ ] Apply for state CSR if required (see [CSR SOP](csr-application.md))

---

## DEA Renewal

- DEA sends renewal reminder 60 days prior
- Renewal fee: $888 for 3 years
- Renewal URL: [dea.diversion.usdoj.gov](https://www.dea.diversion.usdoj.gov/online_forms_and_applications/renewal.htm)
- [ ] Complete renewal, update Provider & Compliance Dashboard
