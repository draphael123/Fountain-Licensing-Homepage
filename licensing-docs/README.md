## Credential Types Covered

- **NP** — Nurse Practitioner (APRN, FNP, PMHNP)
- **MD / DO** — Physician licenses
- **RN** — Registered Nurse (including eNLC compact states)
- **CSR / DEA** — Controlled Substance Registrations & DEA numbers

## Key Tools & Systems

- **CE Broker** — CE tracking and renewal verification
- **Provider & Compliance Dashboard** — Notion-based license tracker
- **Doxy** — Telehealth platform requiring state compliance
- **DocuSign** — Credentialing document execution
