## 90-Day Renewal Pipeline

All licenses expiring within 90 days should be actively in renewal. The [License Expiration Monitor](https://license-expiration-monitor-v2-3ykw-kbikc243m.vercel.app/) provides a real-time dashboard.

**Escalation thresholds:**
- 🟡 90 days — Begin renewal process
- 🔴 30 days — Escalate to Daniel Raphael
- 🔴 Expired — Immediate escalation to CCOO/CMO, provider must stop seeing patients in that state
