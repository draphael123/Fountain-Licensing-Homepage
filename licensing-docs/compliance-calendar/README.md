# Compliance Calendar

This section helps the licensing team stay ahead of expirations, CE deadlines, and renewal windows across all providers and states.

## Overview

The Compliance Calendar has two components:

1. **[Expiration Tracker](expiration-tracker.md)** — All upcoming license, CSR, and DEA expirations sorted by date
2. **[CE Requirements](ce-requirements.md)** — CE hours required per state/credential type and deadlines

## How to Use

- Review this section **weekly** during licensing team standups
- Flag any items expiring within **90 days** for immediate action
- For license renewals, follow the [License Renewal SOP](../sops/license-renewal.md)
- For CE gaps, notify the provider immediately with required hours and recommended courses

## Renewal Lead Times (Policy)

| Document Type | Start Renewal Process |
|---|---|
| State Professional License | 90 days before expiration |
| DEA Registration | 60 days before expiration |
| State CSR | 60 days before expiration |
| Certifications (ANCC, AANP) | 6 months before expiration |
| Collaborating Physician Agreements | 90 days before expiration |

## Color Code System

> 🔴 **Expired or < 30 days** — Immediate action required
>
> 🟡 **30–90 days** — In progress or scheduled
>
> 🟢 **90+ days** — Monitoring
