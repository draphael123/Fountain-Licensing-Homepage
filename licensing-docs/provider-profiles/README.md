# Provider Profiles

This section is for individual provider license tracking. Each provider at Fountain Vitality should have a profile page that the licensing team maintains.

## Purpose

- Central source of truth for each provider's active licenses, expiration dates, and credential status
- Used to cross-reference the Notion Provider & Compliance Dashboard
- Helps identify gaps (missing states, upcoming renewals, missing CSRs)

## How to Use

1. Create one page per provider using the [Provider Profile Template](profile-template.md)
2. Update whenever a license is issued, renewed, or a new state is added
3. Flag providers with licenses expiring within 90 days in the [Compliance Calendar](../compliance-calendar/expiration-tracker.md)
