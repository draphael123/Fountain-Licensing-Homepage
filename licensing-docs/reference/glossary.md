# 📖 Glossary

Key licensing terms, acronyms, and definitions used across this knowledge base.

| Term | Definition |
|---|---|
| **AANP** | American Association of Nurse Practitioners — certifying body for FNPs |
| **ANCC** | American Nurses Credentialing Center — certifies NPs including PMHNP |
| **APRN** | Advanced Practice Registered Nurse — umbrella term covering NPs, CRNAs, CNMs, CNSs |
| **CE / CEU** | Continuing Education (Units) — required hours for license renewal |
| **CPA** | Collaborative Practice Agreement — formal agreement between NP and collaborating physician, required in some states |
| **CSR** | Controlled Substance Registration — state-level registration for prescribing controlled substances |
| **DEA** | Drug Enforcement Administration — federal agency issuing DEA numbers for controlled substance prescribing |
| **DO** | Doctor of Osteopathic Medicine |
| **eNLC** | Enhanced Nurse Licensure Compact — multi-state compact allowing RNs/LPNs to practice under one license |
| **FCVS** | Federation Credentials Verification Service — used for physician credential verification |
| **FNP** | Family Nurse Practitioner |
| **FPA** | Full Practice Authority — NP can practice and prescribe independently without a CPA |
| **GLP** | GLP-1 receptor agonist (e.g., semaglutide) — used for weight management |
| **HRT** | Hormone Replacement Therapy |
| **IMLC** | Interstate Medical Licensure Compact — expedited multi-state licensing for physicians |
| **MD** | Medical Doctor |
| **NP** | Nurse Practitioner |
| **NPI** | National Provider Identifier — 10-digit identifier for all healthcare providers |
| **PAA** | Practice Affiliation Agreement — alternative term for CPA in some states |
| **PDMP** | Prescription Drug Monitoring Program — same as PMP |
| **PMHNP** | Psychiatric Mental Health Nurse Practitioner |
| **PMP** | Prescription Monitoring Program — state database tracking controlled substance prescriptions |
| **RN** | Registered Nurse |
| **Ryan Haight Act** | Federal law requiring in-person evaluation before prescribing controlled substances via telemedicine |
| **SSN** | Social Security Number — sensitive, redacted in this knowledge base |
| **TRT** | Testosterone Replacement Therapy |
