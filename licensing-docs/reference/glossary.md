# Glossary

Key terms and acronyms used across the Fountain licensing knowledge base.

| Term | Definition |
|---|---|
| AANP | American Association of Nurse Practitioners — certifying body for FNPs |
| ANCC | American Nurses Credentialing Center — certifying body for NPs including PMHNP |
| APRN | Advanced Practice Registered Nurse — umbrella term for NPs, CRNAs, CNMs, and CNSs |
| CE / CEU | Continuing Education (Units) — required for license renewal |
| CPA | Collaborative Practice Agreement — formal agreement between NP and collaborating physician required in some states |
| CSR | Controlled Substance Registration — state-level registration required to prescribe controlled substances in many states |
| DEA | Drug Enforcement Administration — federal agency issuing DEA numbers for controlled substance prescribing |
| DO | Doctor of Osteopathic Medicine |
| eNLC | Enhanced Nurse Licensure Compact — allows RNs and LPNs to practice in multiple compact states under one license |
| FCVS | Federation Credentials Verification Service — used for physician credential verification across states |
| FNP | Family Nurse Practitioner |
| FPA | Full Practice Authority — NP can practice and prescribe independently without a CPA |
| GLP | GLP-1 — glucagon-like peptide-1 receptor agonists (e.g., semaglutide) used for weight management |
| HRT | Hormone Replacement Therapy |
| IMLC | Interstate Medical Licensure Compact — expedited multi-state licensing for physicians |
| MD | Medical Doctor |
| NP | Nurse Practitioner |
| NPI | National Provider Identifier — 10-digit number assigned to healthcare providers |
| PAA | Practice Agreement Agreement — alternative term for CPA in some states |
| PDMP | Prescription Drug Monitoring Program — same as PMP |
| PMHNP | Psychiatric Mental Health Nurse Practitioner |
| PMP | Prescription Monitoring Program — state database tracking controlled substance prescriptions |
| RN | Registered Nurse |
| Ryan Haight Act | Federal law requiring in-person evaluation before prescribing controlled substances via telemedicine (with DEA exemptions) |
| SSN | Social Security Number — sensitive field, redacted in this knowledge base |
| TRT | Testosterone Replacement Therapy |
